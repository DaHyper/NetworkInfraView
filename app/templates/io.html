About the raw data format

The JSON/YAML export is designed for AI ingestion and machine parsing. Each file contains a versioned schema with all entities and their relationships. You can feed this directly into a GPT/Claude conversation to ask questions about your infrastructure, generate documentation, or perform analysis. 

{
  "export_version": "1.0",
  "exported_at": "2024-01-01T00:00:00Z",
  "app": "NetworkInfraView",
  "data": {
    "sites": [...],
    "isps": [...],
    "hardware": [...],
    "firewalls": [...],
    "vms": [...],
    "networks": [...],
    ...
  }
}
